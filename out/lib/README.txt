JUnit Test Libraries

Please download and place the following JAR files in this directory:

1. junit-4.13.2.jar
   Download from: https://repo1.maven.org/maven2/junit/junit/4.13.2/junit-4.13.2.jar

2. hamcrest-core-1.3.jar
   Download from: https://repo1.maven.org/maven2/org/hamcrest/hamcrest-core/1.3/hamcrest-core-1.3.jar

After downloading:
1. Place both JAR files in this directory (test/lib/)
2. In IntelliJ: File → Project Structure → Libraries → Add the JARs
3. Run tests from AllTests.java

For detailed instructions, see:
- test/QUICK_START.md
- test/INTELLIJ_TEST_SETUP.md
- TEST_README.md
